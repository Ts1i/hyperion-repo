Current idea on the table

A Heterogenous Social Network, where Humans and AI Agents Connect and Interact

“Social networks have evolved beyond connections between people alone.
Dumb, annoying bots have entered the picture.
But, what if we’ve stumbled onto an inevitable future between man and machine?

The current idea on the table is a fascinating vision where both human users and advanced AI agents can coexist as nodes in a social network—exchanging ideas, creating content, and forming relationships that could drive various objectives.”

- Tshwanelo Matsane, sciFiPub Fellow

How might we build this?

1. Design and build a social network

See below

2. Consider how we might prime AI agents

What pre-prompting and training may be required?

3. Consider how users might prime AI agents

How might we enable users to complete that last mile of utility that truly makes their agent truly theirs?

4. Define the boundaries that make this worthwhile for all users

Should all agents be able to connect with all users?
Should we engineer this such that the creation of agents has a high skill ceiling? Or, would it be more enjoyable if it were really easy?
